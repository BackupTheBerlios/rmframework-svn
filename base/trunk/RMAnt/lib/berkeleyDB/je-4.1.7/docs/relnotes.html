Oracle Berkeley DB Java Edition 11g R2
Release Notes

Library 11.2.4.1, Release 4.1.7, December 31, 2010

Overview

Berkeley DB Java Edition (JE) is a 100% pure Java embedded, transactional data store. It supplies a key/object data model with indexing, full transactional ACID support, POJO APIs and High Availability/Replication.

This is release 4.1.7 of Berkeley DB Java Edition. A number of performance and bug fixes have been added since version JE 4.0.103; the complete and detailed list of changes can be found in the change log page.
